# **App Name**: DevVault Portfolio

## Core Features:

- AI-Powered Content Summarizer: A generative AI tool that automatically condenses project descriptions or learning log entries into concise summaries, making content digestible at a glance.
- Modular Local Content API: Efficiently parse and manage all portfolio content (projects, logs) from local JSON and Markdown files, ensuring type-safe data access across the application.
- Filterable Project Gallery: Showcase all projects and activities in an interactive gallery with filtering capabilities by tech stack or category.
- Dynamic Log & Documentation Pages: Generate individual, SEO-optimized pages for each learning log entry and technical document, leveraging Next.js dynamic routing.
- Interactive Internship Timeline: Present key internship activities and milestones in a visual and interactive timeline format on the home page.
- Responsive & Performant UI: Ensure a highly performant and responsive user experience across all devices, incorporating Next.js Image optimization and dynamic metadata for excellent Lighthouse scores.

## Style Guidelines:

- Primary color: Deep Cyan Blue (#297AA3) for professionalism and a digital aesthetic, signifying technical depth and clarity.
- Background color: Very Light Blue-Grey (#F0F3F5) to provide a clean, expansive, and calming canvas that enhances content readability.
- Accent color: Vibrant Turquoise (#20DFBF) for highlighting interactive elements, calls to action, and key information, creating an energetic yet professional contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and precise feel, reflecting the technical nature of the content. Body font: 'Inter' (sans-serif) for neutral, objective, and highly readable long-form content.
- Code font: 'Source Code Pro' (monospace) for clear and distinguishable display of any code snippets or technical examples.
- Utilize clean, minimalistic line-based icons to align with the modular 'Lego-style' architecture and enhance clarity across the user interface.
- Implement a fluid grid system that seamlessly transitions from a single column on mobile devices to a three-column layout on desktops. Maintain persistent navigation and footer sections.
- Incorporate subtle loading skeletons ('loading.tsx') for smooth content transitions, alongside gentle hover effects for interactive elements to improve user feedback and engagement.
